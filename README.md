基于顺丰开放平台官方SDK的扩展包

## 安装

``` bash
$ composer require jormin/sf-express -vvv
```

## 通用配置

 - appID: 顺丰开放平台App ID
 - appKey: 顺丰开放平台App Key

## 通用响应

| 参数  | 类型  | 是否必须  | 描述  |
| ------------ | ------------ | ------------ | ------------ |
| success | bool | 是 | false：操作失败 true:操作成功 |
| message | string | 是 | 结果说明 |
| data | array | 否 | 返回数据 |
